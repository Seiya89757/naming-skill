# Changelog

本 skill 遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.4.0] - 2026-05-11

「八字」维度的方法论级重构。其余三个维度（家族传承 / 父母期望 / 方言）未改动。

起因：一个真实案例中，专业命理师给出的喜用神与 skill 输出完全相反。复盘发现 skill 的八字推理层级过浅——只做"静态四柱 → 数五行 → 定喜用神"，而命理师做的是"四柱 → 司令分野 → 大运 → 环境 → 区分健康/格局 → 取用神"的多层推理。本版把多层推理补齐。

### Added — 八字方法论
- **§2.0 两套喜忌系统**：区分"格局喜忌（管富贵）"与"健康喜忌（管身心）"，二者可能相反；起名默认服务健康喜忌
- **§2.4 月令人元司令分野**：新增 12 月支司令分野天数表，旺衰判断细分到出生日所在司令段
- **§2.5 大运**：写全大运排法理论（月柱延伸、阳年男/阴年女顺排、3 天折 1 岁起运），以头三步大运修正静态命局
- **§2.6 后天环境五行**：出生地/生活地方位五行（南火北水东木西金中土）纳入用神权重
- **§2.8 冲突处理协议**：模型与命理师结论冲突时的 4 步流程（核对排盘数据 → 分辨分歧类型 → 服从命理师 → 理解 why）

### Added — 立意素材
- 维度 3 新增"命盘结构反推性格"：十神组合与刑冲推导性格倾向，作为起名立意素材，附扬长补短对照表

### Changed
- §2.1 默认立场从"扶抑 + 调候揉一团"改为"格局喜忌走扶抑+格局、健康喜忌走调候+大运修正"，两套分开
- §2.2 用户已提供喜用神时，新增"追问命理师给的是健康还是格局那套"的话术
- §2.4 旺衰判断强制披露置信度，禁止把"偏旺/偏弱"说成定论
- §2.9 字与五行判定从"偏旁 + 字义"升级为三层"偏旁 < 字义 < 意象"，补充无对应偏旁的字也可承载五行能量、"吉利字不分五行通用"
- §2.7 写入"过犹不及"原则——补喜用神/约束忌神都要讲分寸
- 反模式新增 4 条（喜忌混用、旺衰武断、只看静态四柱、只看偏旁）
- Step 1 第 4 问、Step 4 结尾提示、Step 5/6 模板：同步更新为"健康喜忌"框架

### Known limitations
- 大运排法、司令分野模型现在"知道该做"，但精确度仍不及专业排盘工具，§2.5 已要求精确起运交给工具/命理师
- 本版提升的是"知道自己哪里不专业"的能力，不等于八字推算本身变准；命理结论仍以命理师为最终权威

## [1.3.0] - 2026-05-07

### Added — 外挂参考文件
- `references/dialect-pitfalls.md` — 13 种主要方言的具体谐音雷区库（粤、闽南、潮汕、客家、吴、川、湘、赣、晋等），按需加载，不污染主 SKILL context
- `references/avoid-trendy-chars.md` — 时代标签字 / 网红字数据库，按年代分层（2020s 烂大街、2010s 时代感、2000s 显年龄、跨时代经典字）

### Added — 协商模式
- 第三种工作模式：**协商模式**，专门处理夫妻 / 双方父母意见冲突场景
- 输出包含：诉求梳理（区分硬约束 vs 软偏好）、3 类候选（公约数 / 让 A 让步 / 让 B 让步）、可直接使用的家庭沟通话术
- 诚实点出"无解情况"，不强行撮合

### Added — 命理流派立场
- 默认采用扶抑 + 调候二者结合（当代主流命理实践的最大公约数）
- 用户可指定其他流派（格局派、病药派等），承认对小众派别把握不如主流
- 用户已有命理师结论时，**不论流派**都直接采用

### Added — 真太阳时校正
- 出生时间默认按当地（北京时间）填，模型自动按出生地经度做真太阳时校正
- 明示深圳差 24 分、乌鲁木齐差 1.5-2 小时等数据
- 跨时辰会改时柱，影响喜用神

### Added — 用户体验
- 开场能力边界声明（首次进入完整模式时主动展示能做/做不好的事）
- 量化产出小结（Step 5：本次过了 N 候选 / M 维度 / K 方言层）
- 约束清单导出（Step 6：可复制的"我的起名约束"，下次直接粘贴）
- 分享卡片（Step 7：紧凑表格 + 单候选深度卡片两种格式）
- 对话风格原则：术语翻译（对用户用大白话）+ 允许有温度（穿插一两句人话）

### Added — 边界规则
- 生肖宜忌折中立场：用户主动提则采纳，不主动施加，被问则诚实承认主流命理界争议
- 父母同辈字辈冲突规则：字辈按"代"不按"姓氏"，跨代不冲突、同代不重字
- PII 处理声明：信息只本次对话用、不外发、会话结束即丢

### Changed
- description 从 330 字精简到 155 字（-53%），保留触发关键词，加入"夫妻意见不一"等口语场景
- 维度 4 方言扫描从内嵌简列改为"简列 + 引用 references/dialect-pitfalls.md"

### Added — 元数据
- `metadata.version: 1.3.0`
- `license: MIT`
- LICENSE 文件（MIT + 起名场景特殊免责声明）
